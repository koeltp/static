# jsDelivr一个免费的全球CDN
jsDelivr 是一个免费的开源文件内容分发网络 (CDN)。我们与 GitHub 和 NPM 紧密集成，能够自动为几乎所有开源项目提供可靠的 CDN 服务。

我们提供稳定的 CDN，可用于流量巨大的热门网站的生产环境。没有带宽限制或高级功能，任何人都可以完全免费使用。

# 使用方法
https://cdn.jsdelivr.net/gh/user/repo@version/file
- user 是 GitHub 用户名或组织名。
- repo 是仓库名。
- version 可以是分支名、标签或提交哈希（推荐使用标签或提交哈希以保证稳定性）。
- file 是仓库中的文件路径。

如要用logo目录下的text.png图片，可以使用以下地址：  
https://cdn.jsdelivr.net/gh/koeltp/static/logo/text.png
